Exercise Tracker Microservice is a RESTful API that allow the client to create users, get all users, add exercises and retrieve a full/part exercise log of any user, done for FreeCodeCamp APIs and Microservices Course, Project 4







Usage:

POST Create users: POST /api/exercise/new-user

GET All users: GET /api/exercise/users

GET Retrieve a full exercise log: GET /api/exercise/log

GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
